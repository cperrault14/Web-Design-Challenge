Summary Page

This project consists of creating a Python script to visualize the weather of 500+ cities across the world of varying distance from the equator.

To accomplish this, I utilized several Python libraries, and OpenWeatherMap API and my knowledged of Python coding to create 4 plots that compare the Latitude of each city against 4 other variables (Temperature, Humidity, Cloudiness, Wind)
